📊 Dashboard API

REST API for the Business Dashboard — Sales, Purchases, Customers, Vendors, Staff, Bank & Auth.


http://localhost:5000

application/json

Bearer JWT (where applicable)

Auth — /api/auth

▼
POST /api/auth/register Register a new admin user
Request Body
FieldTypeDescription
usernamestringrequiredMin 3, max 100 chars
passwordstringrequiredMin 8 chars
Example
POST /api/auth/register { "username": "admin", "password": "securePass123" }
Responses
201 Created 400 Validation Error 409 User Exists 500 Server Error
POST /api/auth/login Login and receive JWT token
Request Body
FieldTypeDescription
usernamestringrequired
passwordstringrequired
Response Body
{ "token": "eyJhbGci...", "user": { "id": "...", "username": "admin", "role": "admin" } }
Responses
200 OK 401 Invalid Credentials 500 Server Error

Customers — /api/customers

▼
GET /api/customers List all customers
Query Parameters
ParamTypeDescription
namestringoptionalFilter by name (case-insensitive regex)
Responses
200 Customer[] 500 Error
POST /api/customers Create or update a customer
Request Body
FieldTypeDescription
namestringrequiredMax 100 chars. If exists, record is updated.
addressstringoptionalMax 300 chars
phonestringoptionalMax 30 chars
panstringoptionalPAN/VAT number
contact1stringoptionalContact person 1
contact2stringoptionalContact person 2
accountDetailsarrayoptionalLedger rows (sn, invoiceNo, date, programs, debit, bank, acNo, recDate, discount, balance)
Responses
201 Created 200 Updated 400 Validation 500 Error
DELETE /api/customers/:id Delete a customer by MongoDB ID
Path Parameters
ParamTypeDescription
idMongoIDrequiredCustomer's _id
Responses
200 Deleted document 404 Not Found 500 Error
